About the Game

Memory Match is a fast and fun card-flipping game designed to challenge your brain and reflexes. The goal is simple: find matching pairs of cards by flipping two at a time.

Created by: Zakarya Ayyoub Souiki
Purpose: This game was built as a personal project to improve JavaScript skills, explore DOM manipulation, and create an engaging interactive experience for all ages.
